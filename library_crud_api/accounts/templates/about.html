About

About our website

Our website is a CRUD application where you can interact with movies and books. You can browse our database of movies and books, and you can also add new movies and books. All of the information is stored in a SqLite database, and you can access it through the Django admin page.

About the developers

Our website was developed by a team of passionate developers who are learning Django and developing their skills. We are a group of developers, and we worked together to create this project. We hope that you will find it useful and informative.

About the technologies used

Our website uses Django as the framework, with SqLite as the database. We also used CSS to create the responsive design.
